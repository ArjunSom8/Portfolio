## Build-a-Rocket Spacc-GUI Main Rocket Club

An ESP32-based rocket telemetry system with GUI.

- **Location**: `Build-a-Rocket spacc-gui main Rocket Club/`

## Scalable System Monitoring Tool

A microservices monitoring solution using Kubernetes, Terraform, Prometheus, Grafana, and ELK stack.

- **Location**: `Scalable System Monitoring Tool/`
- **Components**:
  - Kubernetes manifests: `k8s/`
  - Helm chart: `helm/mychart/`
  - Prometheus config: `prometheus/`
  - Grafana dashboard: `grafana/`
  - ELK stack: `elk/`
  - Chaos testing: `chaos/`
  - Operator: `operator/`
  - Terraform: `terraform/`
  - Service API: `service/`
